## Auftrag 2
Beantworten sie die folgenden Fragen.

#### *In welche drei Ursachen werden Bedrohungen eingeteilt?*  
Menschliches Versagen, Kriminelle Handlungen und höhere Macht.

#### *Was ist mit Eintrittswahrscheinlichkeit eines Risikso gemeint? Wie hoch schätzen Sie die Möglichkeit ein, dass ein Virus Daten auf ihrem System zerstört? Treffen Sie Massnahmen? Wie gehen Sie dabei vor?*  
Mit der Eintrittswahrscheinlichkeit wird definiert, für wie realistisch man das Szenario hält. Die Möglichkeit besteht. Das wahrscheinlichste Ereignis ist das versehentliche Herunterladen eines Virus. Ich weise diesem Ereignis eine kleine Eintrittswahrscheinlichkeit zu. Es kann durch vermeinden von dubiosen Downloads und separatem Admin auf dem Gerät sehr gering gehalten werden.

#### *Was regelt der Datenschutz? Was bedeuten diese Regelungen für die verantworlichen Informatiker?*
Der Datenschutz definiert per Gesetz, dass sensible Daten einer natürlicher Person nicht an Dritte gelangen darf, so fern es nicht durch die natürliche Person beabsichtigt ist.

#### *Welche Unterschiede zwischen Backup und Archivierung können Sie aufzählen? Kennen Sie Gründe die für die eine oder andere Methode sprechen? Welche würden Sie für ihre persönlichen Daten auswählen?*
Bei einem Backup werden die Daten kopiert, um diese bei einem Datenverlust wiederherzustellen. Bei einem Archiv werden Daten kopiert und auf dem eigentlichen Datenträger gelöscht. Dies, da per Gesetz einige Dokumente über einen gewissen Zeitraum aufbewahrt werden können.
