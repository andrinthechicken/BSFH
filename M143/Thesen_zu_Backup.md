# Thesen zu Backup
### Modul 143

Begründen Sie, weshalb die untenstehenden Aussagen stimmen
oder falsch sind. 

#### These 1:
*"Daten-Backup kostet nichts."*

Backup kostet durchaus. Solche Dienstleistungen sind mit unter sehr kostspielig. So kostet sowohl der Speicher an sich, als auch der Betrieb der Backuplösung. Je nach Anforderungen und Umfang des Backups gehen die Kosten jedoch weit ausseinander. 

#### These 2:
*"Backups sind heute nicht mehr erforderlich."*  

Heute sind Backups wohl so wichtig wie nie zuvor. Immer mehr wichtige und sensible Daten sind heute digital abgelegt. Sollte hier etwas kaputt gehen, ist es verloren. Deshalb müssen diese Information mehrfach gesichert sein.

#### These 3:
*"Daten in der Cloud sind immer sicher."*

Diese Aussage ist nicht korrekt. Auch Daten in der Cloud sind auf irgendeinem Server gespeichert. Diese läuft ebenso wie ein On Premise Server Gefahr, kaputt zu gehen. Die Cloud ist im Prinzip einfach ein Server vor Ort, einfach einen anderen Ort.

#### These 4:
*"Nach dem Einrichten eines Backups ist sichergestellt, dass alle Daten immer gesichert werden."*

Dies stimmt keinesfalls. Selbst wenn Automatismen eingerichtet wurden, muss regelmässig geprüft werden, ob das Backup korrekt angelegt wird und ob aufgrund des verbrauchten Speicherplatzes, eine Auslagerung nötig wäre.

#### These 5: 
*"Der Datenverlust in einer Firma kostet nichts."*

Wenn Daten verloren gehen, sind fast immer relevante darunter. Solche Fälle haben Firmen tagelang lahmgelegt. Zusätzlich fallen oft Kosten durch die IT an.

#### These 6:
*"Es müssen immer alle Daten gesichert werden."*

Veraltete oder unwichtige Daten müssen nicht gesichert werden. Sofern diese keinem weiteren Zweck dienen, hätte ein Verlust keinen Einfluss auf die Firma. Allerdings ist in einem solchen Fall gleich zu einer sofortigen Löschung der Daten zu raten.

#### These 7:
*"Eine Datensicherung pro Woche genügt."*

Diese Aussage kann zutreffen, falls Daten nur selten oder in wöchentlichen Schritten angepasst werden. Ansonsten ist eine regelmässigere Backuplösung zu empfehlen.

#### These 8:
*"Es gibt nur eine Art von Daten. Deshalb kann die Backuperstellung immer gleich ausgeführt werden."*

Es gibt verschiedene Datentypen. Die Backuperstellung muss stehts angepasst werden.

#### These 9:
*"Der Faktor Zeit spielt bei der Backuperstellung keine Rolle."*

Der Zeitfaktor spielt eine grosse Rolle. Werden Daten oft bearbeitet, so gibt es viele Zwischenstände, welche relevant werden könnten. Bei selten bearbeiteten Daten, kann die Spanne grösser sein.

#### These 10:
*"Google macht von den Benutzerdaten keine Backups."*

Google macht durchaus Backups von Benutzerdaten. Benutzerdaten werden gespeichert und sind von einem gewissen Wert. Daher werden sie auch in Backups inkludiert.

#### These 11:
*"Das Erstellen von Backups ist im Geschäftsumfeld freiwillig."*

Diese Aussage ist korrekt, jedoch ist es nicht empfehlenswert, da wichtige Daten verloren gehen könnten.
